# Final Project – Web Development (IBM Full Stack Software Developer)

This repository contains my final project for the course **Introduction to Web Development with HTML, CSS, and JavaScript**.

## Project Overview
The project demonstrates the use of **HTML**, **CSS**, and **JavaScript** to create a simple multi-page personal website that includes:
- A homepage introducing myself
- A skills page describing my technical abilities
- A project page showing my work
- A recommendations page displaying testimonials
- A contact form
